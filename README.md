# Pinch Pita — Landing Page

Премиальный лендинг для проекта автоматизированного фастфуда Pinch Pita.

## Быстрый старт

```bash
# 1. Установите зависимости
npm install

# 2. Запустите локально для проверки
npm run dev

# 3. Соберите для продакшена
npm run build
```

После сборки готовые файлы будут в папке `dist/`.

---

## Деплой

### Вариант 1: Vercel (рекомендуется, бесплатно)

1. Создайте аккаунт на [vercel.com](https://vercel.com)
2. Загрузите проект на GitHub
3. В Vercel нажмите **"New Project"** → выберите репозиторий
4. Vercel автоматически определит Vite и задеплоит
5. Готово — сайт будет доступен по ссылке `your-project.vercel.app`

### Вариант 2: Netlify (бесплатно)

1. Создайте аккаунт на [netlify.com](https://netlify.com)
2. Перетащите папку `dist/` на страницу Netlify
3. Или подключите GitHub-репозиторий для автодеплоя

### Вариант 3: Любой хостинг (VPS, shared hosting)

1. Выполните `npm run build`
2. Загрузите содержимое папки `dist/` на хостинг
3. Настройте веб-сервер (nginx/apache) на папку с файлами

## Структура проекта

```
pinch-pita/
├── index.html          # Точка входа
├── package.json        # Зависимости и скрипты
├── vite.config.js      # Конфигурация сборки
├── public/
│   ├── favicon.svg     # Иконка сайта
│   └── founder.jpg     # Фото для hero-секции
├── src/
│   ├── main.jsx        # Инициализация React
│   └── App.jsx         # Весь лендинг
└── README.md
```

## Как поменять фото

Замените файл `public/founder.jpg` на новое изображение с тем же именем.

## Как поменять видео

В файле `src/App.jsx` найдите секцию `VIDEO` и замените YouTube ID:

```js
{ title: "Презентация продукта", videoId: "74UzAlq2gjU" },
{ title: "Автомат в работе",     videoId: "Xtn4qF6So-U" },
```
